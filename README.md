# Cybersecurity Chatbot - Part 3 (POE)

A WPF-based cybersecurity awareness chatbot with an interactive quiz game.

## Features

- Voice greeting on startup
-  ASCII art logo display
-  Chat-based interface
-  Password safety tips
-  Phishing detection education
-  Safe browsing practices
-  **Interactive Cybersecurity Quiz (NEW in Part 3)**
-  Keyboard shortcuts (Enter to send)

## Quiz Feature

The chatbot includes a 5-question cybersecurity quiz:
- Tests knowledge on phishing, passwords, HTTPS, and 2FA
- Provides immediate feedback on each answer
- Displays final score with encouraging messages

## Requirements

- .NET Framework 4.7.2 or higher
- Windows OS

## How to Run

1. Clone the repository
2. Open in Visual Studio 2022
3. Build the solution
4. Run the application
